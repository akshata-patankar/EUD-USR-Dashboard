# EUD-USR-Dashboard
## EUR/USD Price & Volatility Dashboard

This project presents an interactive financial analytics dashboard analyzing historical **EUR/USD exchange rate** data. The dashboard explores price trends, volatility patterns, and short-term returns to support exploratory financial analysis and market research.

### Tools Used

* Microsoft Power BI
* DAX
* Time Series Analysis

### Features

* 30-day rolling volatility calculation
* 7-day return analysis
* Key KPIs including highest, lowest, and average price
* Interactive filters for exploring historical exchange rate trends

### Use Case
The dashboard enables users to analyze historical currency behavior, monitor volatility trends, and explore price movements of the EUR/USD pair through an interactive BI interface.
